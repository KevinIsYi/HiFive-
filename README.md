# HiFive!

## Description
* This project is an e-commerce website built with the MERN stack.
* Backend is currently deployed at www.heroku.com

## Installation steps

1. `git clone https://github.com/KevinIsYi/HiFive-.git`

### Frontend
1. cd Frontend
2. npm install
3. npm start
4. Enjoy

### Backend
1. cd Backend (from the project's root)
2. npm install
3. npm start
4. Enjoy
